# DILERWIZARTH — Landing (Vite + React + Tailwind)

Proyecto de ejemplo: landing para DILERWIZARTH con tema oscuro y animaciones.

Stack:
- React + Vite
- Tailwind CSS
- Framer Motion
- React Router
- React Icons

Comandos:

```zsh
# instalar dependencias
npm install

# correr en dev
npm run dev

# construir
npm run build

# preview
npm run preview
```

Notas:
- La paleta principal usada: #000000 (fondo), #E63946 (rojo), #C1121F (rojo oscuro), #1A1A1A (gris oscuro), #FFFFFF (texto), #F8F9FA (texto secundario).
- Los productos usan imágenes aleatorias de Unsplash como placeholder.
- Los botones "Consultar Precio" abren WhatsApp con mensaje pre-llenado.
